
<h1>Cbank</h1>

<h4>Descrição</h4>
<p>Um site para você administrar suas criptomoedas, com a posssibilade de adicionar diferentes quantidades na sua carteira e de ver suas informações mais detalhadas,
a aplicação usa a API do CoinGecko.</p>

<b>Para ter acesso a todas as páginas da aplicação é necessário fazer login com uma conta no GitHUb. </b>
